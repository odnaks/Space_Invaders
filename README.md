# Space Invaders
My version of the old school game "Space Invaders"

### how to install:
```shell
git clone https://github.com/odnaks/Space_Invaders.git && cd Space_Invaders && sh INSTALL
```

### how to start:
```shell
sh 1.py
```

```shell
esc or
close
```
